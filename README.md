# Programming Vault

Personal knowledge base for programming, coding, and software architecture concepts.

## Structure

- **Topic files** - Reference notes on programming concepts (lowercase-with-hyphens.md)
- **examples/** - Code snippets and sample implementations
- **diagrams/** - Architecture diagrams and visual references
- **tools/** - Automation scripts and utilities

## Tools & Integrations

- **Obsidian** - Primary note-taking interface with CLI for automation
- **Claude Code** - AI-assisted development and knowledge capture
- **Taskmaster** - Task management for AI-driven development
- **Ralph Loop** - Autonomous AI agent loops
